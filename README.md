# Description
Another multicolors library.
Credits to: multicolors, colors, morecolors, colorlib creators.

## Colors CSS, TF2, DODS, HL2DM
```
{color:default}
{color:team}
{color:red}
{color:blue}
{color:green}
{color:white}
{color:yellow}
{color:purple}
{color:orange}
{color:lime}
{color:grey}
{color:black}
```

You can use any HEX color like this
```
{color:#FFFFFF}
{color:#121212}
{color:#AB89EA}
etc.
```
